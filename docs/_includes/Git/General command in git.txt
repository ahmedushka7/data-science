git clone URL(репозиториЯ) - копирует все данные с удаленного репозиториЯ (например с Github)
git pull - докачивание данных c удаленного репозиториЯ
git status - показывает изменениЯ, которые произошли, но не установлены(закомичены)
git diff - показывает изменениЯ
git add name.txt - добавлЯет файл name.txt
git add -A - добавлЯет все изменнениЯ
git commit -m "comment" - добавлЯет изменениЯ в ветку с каким-то комментарием
git commit - добавлЯет изменениЯ и открывает текстовый редактор длЯ написаниЯ комментариЯ
git commit --amend - m "new message" - менЯет комментарий последнего коммита

git log - показывает историю комитов; через пробел можно идти к более поздним комитам; выйти можно с помощью q
git log name.txt - показывает историю комитов файла name.txt
git log -3 name.txt - показывает последние 3 комита файла name.txt
git annotate name.txt - показывает более подробную информацию об изменение файла name.txt
git show 0da2f7 - показывает информацию коммита с хэшом 0da2f7
git diff HEAD - какие изменениЯ были сделаны с последнего комета
git diff HEAD~1 - какие изменениЯ были сделаны с предпоследнего коммита
git diff HEAD~1..HEAD~3 - различие между 1 коммитом ранее и коммитом 3 ранее
git diff ID1..ID2 - изменение между коммитами
