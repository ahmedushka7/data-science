git clone URL(репозитория) - копирует все данные с удаленного репозитория
git pull - докачивание данных
git status - показывает изменения, которые произошли, но не закомичены
git add -A - добавляет все изменнения
git commit -m "comment" добавляет изменение в ветку
git push - обновление данных на удаленном репозитории
